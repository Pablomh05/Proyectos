# NanoFiles – Sistema P2P en Java

Proyecto desarrollado para la asignatura **Redes de Comunicaciones**.

NanoFiles es un sistema de compartición de archivos **peer-to-peer**, implementado en Java, que permite la localización y descarga de archivos entre nodos distribuidos usando sockets.

## Características
- Arquitectura P2P
- Comunicación mediante sockets TCP
- Registro y localización de archivos
- Transferencia de archivos entre nodos

## Estructura
- src/: código fuente
- jars/: ejecutables
- docs/: documentación y enunciado
- memoria/: memoria técnica
